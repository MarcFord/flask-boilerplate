Flask Boilerplate

Flask Boilerplate is here to help you build a new flask application with support for many useful baked in options!

  • Clean and friendly directory structure
  • SQLAlchemy, Flask-SQLAlchemy, SQLAlchemy-Utils, and a custom base model
  • Alembic (for database migrations)
  • Celery, Flask-Celery, Flask-Celery-Helper, a Custom Task Router
  • Flask-Script; with some custom commands to wrap running gulp tasks and some database commands
  • Flask-DebugToolbar
  • Flask-Classy; with a custom base view class for class based views
  • Flask-Admin
  • WTForms
  • PyYAML, used here for database seeds
  • Gunicorn, for application server with gevent
  • Bootstrap-sass
  • Font-Awesome
